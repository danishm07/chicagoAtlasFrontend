


You are helping build a mobile app called Chicago Pulse 
for a 24-hour hackathon. The AI assistant inside the app 
is named Harold (after Harold Washington Library — a 
Chicago cultural landmark). 

This is a React Native / Expo app. Build mobile-first, 
clean, minimal design. Two modes: dark and light. Default 
to dark on Screen Zero, transition to light after onboarding.

- Light mode: bg #F7F6F3, surface #FFFFFF, text #1C1B18,

  secondary #7C7870, accent #E8533A, border #2A2A2A

- Card radius: 16px
- Accent used sparingly — one element per screen max
- All cards scale(0.97) on press
- Smooth transitions between screens

API INTEGRATION:
POST to: {BACKEND_URL}/api/chat
Body: {
  message: string,
  history: last 6 messages,
  profile: profile from AsyncStorage
}
Response: streaming plain text

Read as stream:
const response = await fetch(BACKEND_URL + '/api/chat', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ message, history, profile })
})
const reader = response.body.getReader()
const decoder = new TextDecoder()
let fullText = ''
while (true) {
  const { done, value } = await reader.read()
  if (done) break
  fullText += decoder.decode(value)
  setCurrentResponse(fullText) // update UI as tokens arrive
}
